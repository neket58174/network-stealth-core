# Комьюнити

Этот репозиторий публичный и развивается вместе с комьюнити.

## Где взаимодействовать

- **Discussions**: вопросы использования, архитектура, roadmap
- **Issues**: воспроизводимые баги и конкретные feature-запросы
- **Pull requests**: точечные изменения кода/документации с зелёным CI

## Формат хорошего issue

Укажите:

- точную команду запуска
- окружение (ОС, shell, виртуализация)
- очищенные от секретов логи
- ожидаемое и фактическое поведение
- минимальные шаги воспроизведения

Не стоит:

- отправлять только скриншоты без текстовых логов
- публиковать секреты (`keys.txt`, приватные ключи, чувствительные ссылки)
- писать общий отчёт "не работает" без контекста

## Формат хорошего PR

- одна задача на один PR
- короткие и предметные commit messages
- docs и tests обновлены вместе с поведением
- CI полностью зелёный до запроса review

Подробно: [.github/CONTRIBUTING.ru.md](../../.github/CONTRIBUTING.ru.md).

## Темы, полезные проекту

- edge-cases установки с воспроизводимым сценарием
- поведение доменного планировщика под реальной нагрузкой
- совместимость клиентов (v2rayN, NekoRay, Sing-box, ClashMeta)
- надёжность rollback и эксплуатационные улучшения
